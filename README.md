# Halyk Travel AI

AI-powered travel planning assistant.

## Stack

Frontend:

* Nuxt 3
* TypeScript

Backend:

* FastAPI
* Python

Data:

* Mock JSON APIs

## Run Backend

```bash
cd backend

source .venv/bin/activate

uvicorn app.main:app --reload
```

Swagger:

http://localhost:8000/docs

## Available APIs

* Flights
* Hotels
* Transfers
* Activities
* Restaurants
* Pharmacy
* Insurance
* Payment
